# Nexus AI Project

## Structure

- **Root/**: Frontend Application (React/Vite)
  - `src/`: Frontend Source Code
  - `index.html`, `vite.config.ts`: Frontend Config
- **backend/**: Backend API (Node.js/Express)

## Quick Start

### Frontend
1. Open terminal in **Root** directory.
2. `npm install`
3. `npm run dev`

### Backend
1. `cd backend`
2. `npm install`
3. `npm run dev`
